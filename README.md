# Simulación lanzamientos de satélites en base a un dataset
Este proyecto es una simulación donde se pueden observar los lanzamientos de satélites a lo largo de los años de diferentes paises.

# Características principales
Lanzamiento de los satélites desde sus respectivos sitios de lanzamientos.
Órbitas realistas calculadas a partir de los datos obtenidos del dataset

# Controles
OrbitControls: Cámara orbital (rota, acerca y aleja con el ratón).
